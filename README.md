# Proyecto de Página de Aterrizaje Estática

Este proyecto es una página de aterrizaje estática.

## Archivos del Proyecto
- `index.html`
- `README.md`
- `LICENSE`

## Instrucciones para Ejecutar Localmente
Para ejecutar este proyecto localmente, simplemente abre `index.html` en tu navegador web favorito.

## Notas de Despliegue
Este proyecto está diseñado para ser desplegado rápidamente en cualquier servidor estático. Simplemente sube los archivos al servidor y asegúrate de que `index.html` sea el archivo predeterminado.

### Importante
El archivo principal es ahora `index.html`, (anteriormente se llamaba `index.html.html`).
